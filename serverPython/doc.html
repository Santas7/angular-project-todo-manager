API Documentation

/api/auth/register

Method: POST

Register a new user.

Request Body:

{
            "email": "user@example.com",
            "password": "password",
            "rememberMe": 1,
            "status": "active",
            "captcha": "captcha_text"
}

/api/auth/login

Method: POST

Login with existing user credentials.

Request Body:

{
            "email": "user@example.com",
            "password": "password"
}

/api/auth/me

Method: GET

Get information about the current user.

Query Parameters:

?id=user_id

/api/auth/logout

Method: POST

Endpoint for logging out of the system.

/api/todo/get-notes

Method: GET

Get all notes for a user.

Headers:

Authorization: user_email

/api/todo/add-note

Method: POST

Add a new note.

Request Body:

{
            "id": 1
            "user_id": 1,
            "title": "Note Title",
            "text": "Note text content",
            "status": "pending",
            "creator: "asd@email.ru
}

/api/todo/delete-note

Method: DELETE

Delete a note.

Query Parameters:

?note_id=note_id

/api/todo/update-note

Method: PUT

Update an existing note.

Request Body:

{
            "id": 1,
            "note_id": 1,
            "title": "Updated Note Title",
            "text": "Updated note text content",
            "status": "completed",
            "creator": "asd@email.com"
}
